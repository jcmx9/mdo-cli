# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Current CalVer release | Yes |
| Older releases | No |

Only the latest release receives security updates.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities via public issues.

Instead:

1. Open a private security advisory on [GitHub](https://github.com/jcmx9/mdo-cli/security/advisories/new).
2. Include steps to reproduce if possible.
3. We will acknowledge receipt within **48 hours**.
4. We aim to provide a fix within **14 days**.

After the fix is released, the vulnerability will be publicly documented.

## Principles

- No retaliation against vulnerability reporters.
- Coordinated disclosure: publication only after a fix is available.
- Credit in the CHANGELOG and release notes (if desired).
